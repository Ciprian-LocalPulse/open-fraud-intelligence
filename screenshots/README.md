# Screenshots

Acest director conține capturi de ecran **anonimizate** cu exemple de fraude documentate.

## Structură

```
screenshots/
├── olx/
├── whatsapp/
├── facebook/
├── instagram/
├── telegram/
├── email/
├── sms/
├── crypto/
├── fake-jobs/
└── fake-shops/
```

## Reguli obligatorii

1. **Anonimizați** orice date personale înainte de a uploada
   - Numere de telefon → blur / pixelate
   - Nume reale → blur / pixelate
   - IBAN-uri, carduri → blur complet
   - Adrese → blur

2. **Format acceptat:** PNG, JPG, WEBP
3. **Dimensiune maximă:** 5 MB per imagine
4. **Denumire:** `[scam-id]-[nr].png` (ex: `olx-0001-01.png`)

## Instrumente recomandate pentru anonimizare

- [Flameshot](https://flameshot.org/) (Linux/Windows/Mac) — blur integrat
- Paint 3D (Windows) — acoperire cu dreptunghi
- Preview (Mac) — adnotare cu forme
- [Canva](https://canva.com) — blur online gratuit
